## What's new

### `!setcost` / `/setcost`

Manually set or clear **FIFO cost lots** (blank deposits, corrections):

```text
!setcost sku=725;6;uncraftable&metal=25&amount=50&mode=replace
!setcost sku=…&clear=true
```

Supports `replace` / `append`, optional sell reprice from the new lot. See the wiki: [Cost Basis & FIFO](https://github.com/uwu6967/tf2autobot-pricedb/wiki/Cost-Basis-and-FIFO).
