## What's new

### рЯУК FIFO cost basis in Discord embeds

Pricelist add / get / update embeds can show **real paid cost** from inventory FIFO:

- Unit count, FIFO / avg / min / max / total
- Suggested **floor sell** using Partial Price Update `minProfitScrap`
- Steam chat `!get` replies include the same cost-basis summary when PPU data exists

### рЯЦ•пЄП Boot-safe panel IPC

While the bot is still starting, panel calls no longer throw cryptic failures:

- `addItem` / `updateItem` / `removeItem` вЖТ clear *"Bot is still starting вАФ try again in a moment"*
- `getPricelist` вЖТ returns `false` until pricelist is ready (panel can retry)

### рЯТ± Manual price turns off that side's live autoprice

When you set a **manual buy** or **manual sell** (commands / slash / panel fields):

- That sideвАЩs `autopriceBuy` / `autopriceSell` is turned **off** unless you explicitly keep it on
- Stops live pricedb from immediately overwriting the price you just typed

### рЯФС Asset-id autoprice lookup

Autoprice refresh looks up prices by the entryвАЩs **item SKU**, not the priceKey вАФ so unique / asset-id keyed items update correctly from the pricer dump.

### рЯІє Cleaner pricelist JSON

`minBuy` / `maxBuy` / `minSell` / `maxSell` are only written when set (no more noisy `null` range fields).

Discord embeds also show buy/sell **price ranges** when configured.
