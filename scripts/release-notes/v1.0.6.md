# v1.0.6

## What's new

- **Upstream 5.17.0 RC** merged into the uwu6967 blank-bot fork while keeping Discord Autokeys, partial autoprice, and listing UX.
- **Panel IPC** continues to emit readable string errors for add/update/remove (compatible with [tf2autobot-gui-panel](https://github.com/uwu6967/tf2autobot-gui-panel) v3.6.2).
- **Blank install** — clone without secrets: copy `template.env` / `.env.example` to `.env`; `files/` stays local and gitignored.

## Compatibility

Tested with **tf2autobot-gui-panel v3.6.2**.

## Update

```bash
git pull
npm install
npm run build
```

Or run `!updaterepo` in Steam chat if you use PM2.
