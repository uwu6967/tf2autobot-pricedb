# v1.0.9 — Pure Hive moved to a separate fork

**Pure Hive is no longer in this repo.**

| Repo | Role |
|---|---|
| **[tf2autobot-pricedb](https://github.com/uwu6967/tf2autobot-pricedb)** (this) | Blank trading bot |
| **[tf2autobot-pricedb-hive](https://github.com/uwu6967/tf2autobot-pricedb-hive)** | Same bot **+ Pure Hive** |
| **[tf2autobot-pure-hive](https://github.com/uwu6967/tf2autobot-pure-hive)** | Hive API service |

If you need keys/ref sharing between bots, switch to:

```bash
git clone https://github.com/uwu6967/tf2autobot-pricedb-hive.git
```

and run the [Hive API](https://github.com/uwu6967/tf2autobot-pure-hive).

## Upgrade (stay on blank bot)

```bash
git pull
npm install
npm run build
pm2 restart <your-bot>
```

Or `!updaterepo`. Remove any `HIVE_*` env vars / `hive` options if you had them — they are unused here now.

## Compatibility

Panel: [tf2autobot-gui-panel v3.6.2](https://github.com/uwu6967/tf2autobot-gui-panel/releases/tag/v3.6.2)
