Opt-in **keys & refined sharing** between your own fork bots. Mutual link required. Off by default.

**Pair with:** [tf2autobot-gui-panel **v3.6.2**](https://github.com/uwu6967/tf2autobot-gui-panel/releases/tag/v3.6.2)  
**Hive API:** [uwu6967/tf2autobot-pure-hive](https://github.com/uwu6967/tf2autobot-pure-hive)

---

## What's new

### Pure Hive

Move Mann Co. keys and refined metal between linked bots running this fork:

- **Mutual link** (invite + accept) before any pure moves
- **Auto-rebalance** toward min/max bands (optionally from Autokeys bands)
- **Manual push** via Steam / Discord
- Caps: `maxKeysPerTransfer`, `maxRefinedPerTransfer`, `cooldownSeconds`
- Auto-accept inbound hive offers only from linked partners

### Commands

| Command | Meaning |
|---|---|
| `!hive status` | Heartbeat + status |
| `!hive enable` / `disable` | Toggle in options |
| `!hive link <steamid64>` | Invite |
| `!hive accept <steamid64>` | Accept invite |
| `!hive unlink <steamid64>` | Break link |
| `!hive push keys=N&ref=N&to=<steamid64>` | Manual send |
| `!hive bots` | Directory of hive bots |

Same actions available as Discord slash `/hive`.

### Setup (blank)

1. Clone & run the API: https://github.com/uwu6967/tf2autobot-pure-hive  
2. `npm run token -- "bot-label"` → put token in bot `.env` as `HIVE_TOKEN`  
3. Set `hive.enable=true` in `options.json` (or `!hive enable`)  
4. Link bots with `!hive link` / `!hive accept`

Full guide: [wiki/Pure-Hive.md](https://github.com/uwu6967/tf2autobot-pricedb/blob/master/wiki/Pure-Hive.md)
