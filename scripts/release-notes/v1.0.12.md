## What's new

Trade offer summaries (Discord / Steam chat) now include:

- **Keys used** — how many Mann Co. keys each side gave
- **Worth** — metal total ≈ keys at the offer key rate (or current key price)

Same value both sides → one worth line; otherwise each side is listed.
